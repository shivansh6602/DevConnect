# DevConnect

DevConnect is a full-stack developer social network where developers can create profiles, share posts, and connect with other developers.

## Tech Stack

Frontend:
- React
- Vite
- React Router

Backend:
- Node.js
- Express.js

Database:
- MongoDB

## Features

- User authentication
- Developer profiles
- Post creation
- Developer feed
- Follow developers
